# AtliQ Hospitality Dashboard

## Project Overview

This Power BI dashboard analyzes hotel performance for AtliQ Hospitality from **May 2022 to June 2022**.

The dashboard provides insights into revenue, occupancy, ADR, RevPAR, realization rate, booking trends, and property-level performance.

---

## Dashboard Pages

### Executive Overview

![Executive Overview](Images/executive-overview)

Key metrics:

- Revenue
- RevPAR
- ADR
- Occupancy %
- Realization %
- DSRN

Visualizations include:

- Revenue by city
- Weekly occupancy trend
- Average rating
- Booking platform contribution
- Occupancy by day type

---

### Property Analysis

![Property Analysis](Images/property-analysis)

Provides detailed property-wise analysis including:

- Revenue
- RevPAR
- Occupancy
- ADR
- DSRN
- DBRN
- DURN
- Realization %
- Cancellation %
- Average Rating

Conditional formatting highlights top and low performing properties.
